# 🧩 Sudoku Solver (C++)

## 📌 Overview

A Sudoku solver using backtracking algorithm to solve 9x9 puzzles.

---

## ⚙️ Features

* Solves Sudoku
* Backtracking algorithm
* Input validation
* Demo puzzle

---

## ▶️ How to Run

### 🔹 Run using EXE

Double-click:
sudoku_solver.exe

### 🔹 Run using Compiler

g++ sudoku_solver.cpp -o sudoku
./sudoku
